refreshAccessToken()method 
public function refreshAccessToken():void

To refresh the access token, call the refreshAccessToken function. This method can be used up to 10 days after the access token has expired.

If the method fails, you must prompt the user to log in again.

If the call succeeds, a new access token is available through the getAccessToken function.

This method will dispatch one of two events:

  • LineSDKEvent.REFRESH_TOKEN_COMPLETE: if the token was refreshed;
  • LineSDKEvent.REFRESH_TOKEN_ERROR: if there was an issue refreshing the token. Check the event details for more information on errors.

verifyAccessToken()method 
public function verifyAccessToken():void

To verify the validity of the access token, call the verifyAccessToken function.

This method will dispatch one of two events:

  • LineSDKEvent.VERIFY_TOKEN_COMPLETE: if the token is valid;
  • LineSDKEvent.VERIFY_TOKEN_ERROR: if the token is invalid or if there was an issue accessing the API. Check the event details for more information on errors.
